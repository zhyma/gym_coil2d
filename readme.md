## Setup

You need to have `gym` and `box2d-py` installed. 
1. Setup [Swig](http://www.swig.org/download.html) first if you don't have it installed.
2. install Pyglet by:
        `pip install pyglet`
3. `box2d-py` can be installed seperately or by calling
    `pip install gym[box2d]`
